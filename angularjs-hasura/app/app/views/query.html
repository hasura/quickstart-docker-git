Examples of using Hasura Data APIs

  • Create a new table called "todo" using Hasura Console, and add the following columns:
    • id : Integer (auto-increment)
    • title : Text
    • user_id : Integer
    • completed : Boolean
  • Select 'id' as the Primary Key, click Create
  • Goto Permissions tab, click on 'Add permissions for a new role'.
  • Select User from the dropdown
  • Enter 
    {"user_id": "REQ_USER_ID"}
    in all the Check and Filter fields.
  • Toggle All for Select
  • Tick completed on Update section
  • Save changes
